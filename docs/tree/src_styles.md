# TREE_MAP src_styles

对应真实目录：`/src/styles`

app.css —— 样式入口文件，只负责样式分片导入顺序（修改影响样式加载）
base.css —— 基础样式文件，只负责全局渲染基线（修改影响全局基线）
layout.css —— 布局样式文件，只负责应用结构布局（修改影响布局）
buttons.css —— 按钮样式文件，只负责按钮视觉状态（修改影响按钮 UI）
forms.css —— 表单样式文件，只负责输入控件视觉（修改影响表单 UI）
translation.css —— 翻译页样式文件，只负责翻译页面视觉（修改影响翻译 UI）
settings.css —— 设置页样式文件，只负责设置表单视觉（修改影响设置 UI）
feedback.css —— 反馈文本样式文件，只负责提示文本视觉（修改影响提示 UI）
history.css —— 历史页样式文件，只负责历史列表视觉（修改影响历史 UI）
