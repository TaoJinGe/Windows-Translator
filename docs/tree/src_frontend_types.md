# TREE_MAP src_frontend_types

对应真实目录：`/src/frontend/types`

settings.ts —— 设置类型和默认值，负责前后端 AppSettings 契约（修改影响配置结构）
translation.ts —— 翻译请求、响应和流式事件类型，负责翻译 API 类型约束（修改影响翻译调用）
history.ts —— 历史记录类型，负责历史数据展示契约（修改影响历史 UI）
models.ts —— 服务商和模型选项列表，负责模型下拉选择数据（修改影响模型选择）
languages.ts —— 语言和默认语言对选项，负责翻译方向选择数据（修改影响语言选择）
