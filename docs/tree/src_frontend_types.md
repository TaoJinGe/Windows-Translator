# TREE_MAP src_frontend_types

对应真实目录：`/src/frontend/types`

settings.ts —— AppSettings 类型和默认值，负责设置结构与默认互译语言（修改影响前后端配置契约）
translation.ts —— 翻译请求/响应类型，负责翻译服务契约（修改影响 API 调用）
history.ts —— 历史记录类型，负责历史数据结构（修改影响历史显示）
models.ts —— 模型公司和模型列表，负责设置页下拉选项（修改影响模型选择）
languages.ts —— 语言和互译语言对列表，负责翻译首页语言下拉与设置页默认互译选择（修改影响翻译方向选择）
